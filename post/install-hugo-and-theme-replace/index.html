今天尝试使用Hugo+Github Pages搭建个人博客。

安装Hugo

这里介绍Mac和Windows两种系统对应的软件安装。其他请参考大神Hugo+Github Pages，感谢分享。

Mac 安装Hugo

既可以使用 HomeBrew 安装，也可以在 github.com/gohugoio/hu… 下载二进制包安装

1
2

# HomeBrew安装
brew install hugo

Windows 安装Hugo

在 github.com/gohugoio/hu… 找到32位或64位windows对应压缩包（如我的用了hugo_0.79.0_Windows-64bit.zip），下载二进制包安装。

解压后，有一个hugo.exe文件，把该文件目录加入到环境变量Path中

命令行输入hugo version ,输出版本信息表示安装成功。更多安装方式https://gohugo.io/getting-started/installing

建立Hugo项目

一个 Hugo 项目就是一个站点，创建命令如下：

1
2

hugo new site [project-name]

例如我的站点名称是 blog,创建命令如下：

1
2

hugo new site blog

创建完成后，在 blog 文件夹下会生成以下文件结构：

├── layouts # 存放布局文件
├── static # 存放静态文件 图片 CSS JS文件
└── themes # 存放主题

添加主题

为了快速搭建博客，可以使用主题。使用主题后，只需要向 content 文件夹添加 Markdown 文件即可。

Hugo 有主题市场 themes.gohugo.io/ ，好看的主题：

  1. github.com/olOwOlo/hug…
  2. github.com/yoshiharuya…

第一个好看一些，我选择了第一个。进入根目录，克隆主题文件就是安装主题。

1
2

cd blog
git clone https://github.com/olOwOlo/hugo-theme-even themes/even

进入 blog/themes/even/exampleSite 文件夹，将 config.tom 文件拷贝到项目根目录下，同时将 blog/themes/even/exampleSite/content 文件夹覆盖掉根目录下的 content 。

其中config.tom文件是配置文件，根据自己的想法更改。

启动Hugo

命令行输入以下命令，启动 Hugo :

1

hugo server

在浏览器打开 http://localhost:1313/ 即可查看效果。

为了个人使用，需要修改 config.tom 文件，根据说明修改配置即可。

添加新博客文章

使用hugo命令 添加新博客模板，命令如下：

1

hugo new post/my-first-blog.md

打包

为了部署到Github Pages 需要将Markdown文件打包成HTML文件，打包命令如下：

1
2

# even是主题
hugo -t even

部署到Github Pages上

打包之后就是纯 HTML 文件，理论上所有支持部署静态页面的网站都是支持的。

#!/bin/bash
# 部署到 github pages 脚本
# 错误时终止脚本
set -e

# 删除打包文件夹
rm -rf public

# 打包。even 是主题
hugo -t even # if using a theme, replace with `hugo -t <YOURTHEME>`

# 进入打包文件夹
cd public

# Add changes to git.

git init
git add -A

# Commit changes.
msg="building site `date`"
if [ $# -eq 1 ]
  then msg="$1"
fi
git commit -m "$msg"

# 推送到github
# git push origin master 只能使用 master分支

# 回到原文件夹
cd ..

#!/bin/sh

cd E:/Personal/hugoSrc # hugo site生成的目录

var1=$(pwd)
echo "1. The current working directory $var1."

# If a command fails then the deploy stops
set -e

printf "\033[0;32mDeploying updates to GitHub...\033[0m\n"

rm -rf public

# Build the project.
hugo -t even # if using a theme, replace with `hugo -t <YOURTHEME>`

# copy public folder content to github pages folder
cp -R ./public/* E:/Personal/martin6699s.github.io/

cd E:/Personal/martin6699s.github.io/ # gihub pages 目录

var2=$(pwd)
echo "2. The current working directory $var2."


# Add changes to git.
git add .

# Commit changes.
msg="rebuilding site $(date)"
if [ -n "$*" ]; then
	msg="$*"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master
